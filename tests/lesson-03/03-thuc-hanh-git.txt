a. git init
=> tất cả file trong folder 'demo' đều nằm ở Working Directory

b. tạo file file1.txt, file2.txt, oops/file3.txt
=> Cả 3 files đều nằm ở Working Directory

c. thêm file .gitignore, thêm 2 dòng:
 i. file1.txt
 ii. oops/
=> Cả 3 files đều nằm ở Working Directory

d. Chạy lệnh: git add .
=> file2.txt được add vào Staging Area
=> file1.txt và oops/file3.txt còn nằm ở Working Directory

e. chạy lệnh: git commit -m "init project"
=> file2.txt được chuyển đến Repository
=> file1.txt và oops/file3.txt còn nằm ở Working Directory
