Giải Thích câu lệnh git:
- git init: đưa file từ local vào working directory
- git add <file_name>: đưa <file_name> vào Staging area
- git add .: đưa tất cả file và folder vào Staging area (ngoại trừ các files và folders được quy định trong '.gitignore')
- git commit -m "<message>": comment <message> vào file nằm trong Staging area trước khi đưa lên Repo
