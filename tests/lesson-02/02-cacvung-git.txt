1. Tạo 3 files: file1, file2, file3
=> Các files đang nằm ở vùng local

2. Chạy lệnh: git init
=> Đưa cả 3 files vào vùng Working directory

3. Chạy lệnh: git commit -m "init project"
=> Hệ thống sẽ thông báo là chưa có file nào trên working tree
=> 3 files vẫn nằm trên working directory

4. Chạy lệnh: git add file1
=> file1 được đưa vào Staging area
=> file2 và file3 vẫn nằm ở Working directory

5. Chạy lệnh: git commit -m "add file1"
=> Comment vào file1, file1 vẫn nằm ở Staging area
=> file2 và file3 vẫn nằm ở Working directory

6. Chạy lệnh: git commit -m "add file"
=> Hệ thống ra thông báo tương tự với số 3
=> file1 vẫn nằm ở Staging area
=> file2 và file3 vẫn nằm ở Working directory
